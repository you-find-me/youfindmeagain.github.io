docker导入导出保存加载

一、镜像的导入与导出：
import 与export 命令通常用于Docker镜像的导入导出。
1、将镜像导出到文件：把某一容器当时的状态打包成镜像
docker export 已经存在的镜像id或者镜像名称 > 导出的镜像路径和文件名称.tar
docker export cbe3cb7799ed > test1.tar
2、基于导出的文件导入一个新镜像：把容器快照导入镜像库，可指定url导入
docker import – 新镜像的路径和名称<导出的镜像名称.tar
docker import – test1 < test1.tar

二、镜像的保存与加载：
save 和 load 命令，常用于镜像的保存与加载。
1、保存镜像：将一个镜像父层标签所有历史信息都保存
docker save -o 保存后镜像的路径和名称.tar 已经存在的镜像id或者镜像名称
docker save -o test2.tar test2
2、加载镜像：将镜像存储文件保存到镜像库中，不能通过指定url导入
docker load < 要导入的镜像名称.tar
docker load < test2.tar
